# StudentConnect

A web platform connecting university students in Kosovo with affordable housing options and compatible roommates.

## Tech Stack

- **Frontend:** React.js (Vite)
- **Backend:** Node.js + Express.js
- **Database:** PostgreSQL

## Getting Started

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
